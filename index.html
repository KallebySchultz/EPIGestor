Sobre o Sistema

Funcionalidades Principais

  • Cadastro de EPIs: Gerencie equipamentos com informações completas incluindo validade, quantidade mínima e classificação
  • Controle de Estoque: Monitore níveis de estoque com alertas automáticos para reposição
  • Movimentações: Registre entradas, retiradas, devoluções e descartes com histórico completo
  • Funcionários: Cadastre funcionários e acompanhe o histórico de uso de EPIs
  • Alertas Visuais: Notificações para estoque baixo, vencimento próximo e EPIs vencidos

Características do Sistema

  • Interface Responsiva: Funciona perfeitamente em computadores, tablets e smartphones
  • Design Intuitivo: Navegação simples e clara com cores azul e branco
  • Segurança: Controle de acesso diferenciado para administradores e usuários
  • Relatórios: Histórico detalhado de todas as movimentações
  • Manutenção Fácil: Código bem documentado e estruturado

Requisitos Técnicos

Tecnologias Utilizadas:
• Frontend: HTML5, CSS3, JavaScript
• Backend: PHP (apenas para conexão com banco)
• Banco de Dados: MySQL
• Design: Responsivo, compatível com todos os dispositivos
Configuração Inicial:
1. Configure o banco de dados MySQL executando o arquivo database.sql
2. Ajuste as configurações de conexão em config/database.php
3. Acesse o sistema através do painel administrativo ou de usuário
