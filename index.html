Sobre o Sistema

Funcionalidades Principais

  • Interface Estilo Planilha: Listas editáveis simples e diretas para gestão de EPIs e funcionários
  • Gestão Centralizada: Tudo é administrado centralmente, sem complexidade de múltiplos usuários
  • Login Simples: Apenas um sistema de login para controle de acesso
  • Movimentações Básicas: Registre retiradas, devoluções e ajustes de estoque facilmente
  • Sem "Firulas": Interface limpa, sem funcionalidades desnecessárias

Características do Sistema

  • Interface Estilo Planilha: Familiar e fácil de usar, como uma planilha eletrônica
  • Design Simples: Cores azul e branco, navegação clara e direta
  • Acesso Controlado: Login simples para múltiplas pessoas com acesso
  • Listas Editáveis: Edite diretamente nas tabelas, como numa planilha
  • Zero Complicação: Apenas as funcionalidades essenciais, nada mais

Requisitos Técnicos

Tecnologias Utilizadas:
• Frontend: HTML5, CSS3, JavaScript
• Backend: PHP (apenas para conexão com banco)
• Banco de Dados: MySQL
• Design: Responsivo, compatível com todos os dispositivos
Configuração Inicial:
1. Configure o banco de dados MySQL executando o arquivo database.sql
2. Ajuste as configurações de conexão em config/database.php
3. Acesse o sistema através do painel administrativo ou de usuário
